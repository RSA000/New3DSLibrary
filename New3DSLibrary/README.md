Online Library for 3DS
----------------------------------



## css
    
    * ds.css - Provides base CSS for Nintendo DS system browsers.
    * desktop.css - Provides base CSS for regular Desktop browsers.
    * Other files are theme files that provide additional styling.

## img
    
    * backgrounds contain images for themes that use images for backgrounds.
    * favicon contains site favicon image.
    * gifs contain gif photos.
    * index contains photos used for index page graphics.
    * photos contains photos for the gallery page.

## js
    
    * common.js contains functionalities that are used across all pages.
    * catalogue.js contains functionalities for loading the catalogue page.
    * gallery.js contains functionalities for navigating and loading the gallery page.
    * read.js contains additional control and parsing functionalities for reading literature.

## texts
    
    * texts contains files for literature and literature photos.
